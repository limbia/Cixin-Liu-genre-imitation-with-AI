# Plasticity Radial Menu Editor 离线版

此目录包含从 [PepperKUN/plasticity-radial-menu-editor](https://github.com/PepperKUN/plasticity-radial-menu-editor) 构建的离线版本。要在内网或无外网环境下使用，只需打开 `plasticity-radial-menu-editor/index.html` 即可，界面与官方演示页面保持一致。

> 提示：建议使用现代浏览器（如 Chrome、Edge 或 Firefox）直接打开文件，所有脚本、样式和字体资源均已本地化，无需联网。
