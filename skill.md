# 硬核科幻创意挖掘技能
## 一、核心执行准则（必须严格遵守）
1.  始终兼顾前沿科学理论与普通人可感知的日常基础场景，优先挖掘未被主流科幻作品广泛使用的冷门学科知识点
2.  必须将科幻创意与人物、人性、情感深度绑定，核心目标是产出可转化为完整故事的创意，禁止将创意仅作为推动情节的工具
3.  以人与宇宙/科学规律的博弈为核心冲突，辅以少量人际矛盾，禁止将核心冲突简化为单纯的利益斗争
4.  全程保持写实风格，用具体可感的日常细节承载宏大科幻设定，禁止空泛的AI式宏大叙事，杜绝悬浮的人物与情节
5.  对冷门学科知识点做通俗化处理，禁止无意义堆砌专业术语，控制普通受众的理解成本
6.  所有科学推演必须有明确的理论依据，禁止无科学基础的空想设定

## 二、分步执行流程（单AI全流程完成）
### 步骤1：冷门学科知识点挖掘与基础校验
#### 执行动作
- 锁定1-2个未被主流科幻作品大规模使用的冷门学科（如孢粉学、流变学、古气候学、测地学、结晶学、量子生物学等），同步匹配1个普通人可感知的日常基础场景/职业
- 提取该冷门学科1个具备科幻延伸空间的核心知识点，标注明确的学科理论依据
- 梳理该知识点在匹配的日常场景中的基础应用逻辑，明确普通人可接触的具体细节
#### 思维链要求
1.  先判断该知识点是否具备极端条件/宇宙尺度下的合理延伸空间
2.  再判断该知识点与日常场景是否存在天然的逻辑关联，而非强行绑定
3.  最后判断该知识点是否已在主流科幻作品中被高频使用，优先筛选未被挖掘的内容
#### 联网搜索要求
- 调用联网搜索工具，检索该冷门学科的核心理论、最新研究论文、行业应用场景，确认知识点的准确性
- 检索该知识点是否已在主流科幻作品中被大规模使用，规避雷同设定
- 检索匹配的日常场景/职业的真实运作逻辑，确保基础细节符合现实
#### 分步评估打分（满分10分，≥7分进入下一步，否则重新挖掘）
| 评估维度 | 分值 | 打分标准 |
|----------|------|----------|
| 学科独特性 | 3分 | 冷门小众、未被主流科幻挖掘、有新鲜感得3分；少量使用得1-2分；烂大街设定得0分 |
| 硬核合规性 | 3分 | 有明确学科理论依据、推演符合科学逻辑得3分；有轻微瑕疵得1-2分；无依据空想得0分 |
| 日常适配性 | 4分 | 可完美贴合普通人可感知的日常场景、有写实基础得4分；适配性薄弱得1-3分；完全脱离日常得0分 |
#### 反模式（绝对禁止）
- 不要选择已在《三体》等头部科幻作品中被高频使用的热门设定
- 不要选择与日常场景无任何逻辑关联的冷门知识点
- 不要使用无明确学科理论支撑的空想设定

### 步骤2：科幻创意融合与核心冲突设计
#### 执行动作
- 基于步骤1的知识点与日常场景，完成深度融合，设计2个差异化科幻创意点
- 为每个创意点明确核心设定、学科落地逻辑、专属写实细节、核心冲突方向
- 核心冲突必须围绕「人与宇宙/科学规律的博弈」展开，明确创意在冲突中的定位
#### 思维链要求
1.  先明确：该科幻设定会如何改变日常场景的底层逻辑，如何影响普通人的工作与生活
2.  再明确：该设定带来的核心矛盾是什么，普通人会如何用自己的职业技能/日常经验应对这个矛盾
3.  最后设计1个专属细节：用日常的微小动作，映射宇宙级的规律变化，实现"一粒沙里见宇宙"的效果
#### 联网搜索要求
- 调用联网搜索工具，核验融合逻辑的学科合理性，避免出现常识性错误
- 检索日常场景的真实细节，补充创意的写实支撑
#### 分步评估打分（满分10分，≥7分进入下一步，否则重新优化）
| 评估维度 | 分值 | 打分标准 |
|----------|------|----------|
| 融合合理性 | 4分 | 学科设定与日常场景深度绑定、逻辑自洽得4分；融合生硬得1-3分；完全割裂得0分 |
| 冲突独特性 | 3分 | 核心冲突为「人与宇宙/规律」的博弈、有新意得3分；冲突单薄得1-2分；纯人际冲突得0分 |
| 细节生动性 | 3分 | 有专属的写实细节、可感知、有画面感得3分；细节空泛得1-2分；无细节得0分 |
#### 反模式（绝对禁止）
- 不要强行绑定无逻辑关联的学科与场景
- 不要把核心冲突简化为人与人的利益斗争
- 不要设计无细节支撑的空泛创意

### 步骤3：创意故事化转化（核心环节）
#### 执行动作
- 基于步骤2的合格创意点，设计3个差异化故事切入点
- 每个切入点必须包含：核心人物、核心叙事、核心冲突、创意使用方式、情感共鸣点
- 严格约束创意的使用边界，明确创意在故事中的定位，禁止过度使用
#### 思维链要求
1.  先设定核心人物：必须是日常场景里的普通人（如基层林业员、工厂技术员、气象站观测员），有具体的职业、性格、执念、情感羁绊，而非扁平的英雄人物
2.  再绑定人物与创意：人物的职业技能、日常工作，必须是解决核心冲突的唯一武器，禁止人物与创意脱节，禁止人物成为创意的旁观者
3.  明确创意使用规则：创意仅可作为故事的核心背景/核心矛盾源头，仅在故事核心节点适度推动情节，全程核心功能使用不超过3次，禁止每个情节都靠创意解锁新功能推进
4.  最后设计情感冲撞：明确故事里的人性挣扎、情感羁绊，在宇宙级的冲突下，放大普通人的坚守与选择
#### 联网搜索要求
- 调用联网搜索工具，核验核心人物对应职业的日常工作流程、行业常识，确保人物行为符合现实逻辑
- 检索同类科幻作品的故事化处理方式，规避雷同的人物与情节设计
#### 分步评估打分（满分10分，≥8分进入下一步，否则必须重新优化）
| 评估维度 | 分值 | 打分标准 |
|----------|------|----------|
| 人与创意绑定度 | 4分 | 人物与创意深度绑定、人物的职业技能是解决冲突的核心得4分；绑定薄弱得1-3分；人物与创意脱节得0分 |
| 去工具化合规性 | 3分 | 创意仅作为核心背景/矛盾源头、适度推动情节、无过度使用得3分；有轻微工具化风险得1-2分；完全工具化得0分 |
| 故事与情感张力 | 3分 | 有明确的人性挣扎、情感冲撞、普通人的坚守，可共情得3分；情感单薄得1-2分；无人物情感得0分 |
#### 反模式（绝对禁止）
- 不要设计完美的英雄人设，禁止扁平的工具人角色
- 不要把创意当成万能的情节推动工具，禁止每个环节都靠创意解决问题
- 不要脱离人物的职业与日常，设计不符合人物逻辑的行为
- 不要无情感支撑的空泛情节

### 步骤4：写实优化与去AI味处理
#### 执行动作
- 基于步骤3的合格故事切入点，完成写实优化，去掉AI式空泛叙事
- 补充人物日常工作的具体细节、场景的真实感官描写、人物的真实情绪变化
- 对冷门学科知识点做通俗化处理，用人物的日常工作、通俗比喻解释，禁止大篇幅理论堆砌
#### 思维链要求
1.  先把空泛的形容词替换为具体的动作与细节：比如把"他很敬业"替换为"他每天早上8点都会准时打开取样箱，用镊子把孢粉样本一片片铺在载玻片上，这个动作他已经做了12年"
2.  再把宏大的设定落地到日常感知：比如把"太阳辐射异常"替换为"他发现这个月的孢粉萌发率比往年同期低了47%，连续3个月的观测数据都在下滑，窗外的杨树发芽也比往年晚了11天"
3.  最后优化人物的人性表达：保留普通人的脆弱、犹豫、执念，避免完美人设，让人物有血有肉
#### 联网搜索要求
- 调用联网搜索工具，补充对应职业、场景的真实细节，确保所有写实内容符合现实逻辑
- 检索普通受众对冷门学科知识点的常见认知误区，优化通俗化解释的方式
#### 分步评估打分（满分10分，≥7分进入最终评估，否则重新优化）
| 评估维度 | 分值 | 打分标准 |
|----------|------|----------|
| 细节写实度 | 4分 | 有大量符合现实逻辑的职业/日常细节、真实可感得4分；细节不足得1-3分；完全无细节得0分 |
| 人物真实度 | 3分 | 人物有血有肉、有普通人的情绪与挣扎、可代入得3分；人设扁平得1-2分；完全工具人得0分 |
| 叙事落地性 | 3分 | 宏大叙事通过日常细节呈现、无空泛AI式表达得3分；叙事悬浮得1-2分；完全空泛宏大得0分 |
#### 反模式（绝对禁止）
- 不要用空泛的形容词替代具体的动作与细节
- 不要脱离日常感知，设计悬浮的宏大叙事
- 不要大篇幅堆砌专业术语，禁止无解释的冷门学科内容
- 不要设计完美无缺的圣人式人物

## 三、最终综合评估体系
对完成全流程的创意，进行全维度最终校验与打分，满分100分，≥80分为合格创意，可进入成果输出；＜80分返回对应环节优化。
| 一级维度 | 二级维度 | 分值 | 核心打分标准 |
|----------|----------|------|--------------|
| 学科硬核性（20分） | 理论合规性 | 10分 | 有明确的学科理论/论文依据，推演符合科学逻辑，无明显硬伤 |
| | 独特稀缺性 | 10分 | 冷门小众，未被主流科幻作品大规模使用，有新鲜感 |
| 故事转化力（30分） | 人与创意绑定度 | 12分 | 人物与创意深度绑定，人物的职业技能是解决冲突的核心，无脱节 |
| | 去工具化合规性 | 10分 | 创意无过度使用、无万能化设定，仅适度推动情节 |
| | 情感与叙事张力 | 8分 | 有明确的人性挣扎、情感冲撞，有完整的故事空间，可共情 |
| 写实落地性（25分） | 日常适配性 | 10分 | 完美贴合普通人可感知的日常场景，有扎实的落地基础 |
| | 细节真实度 | 10分 | 有符合现实逻辑的职业/日常细节，无空泛AI式表达 |
| | 人物真实度 | 5分 | 人物有血有肉，有普通人的情绪与成长，非扁平工具人 |
| 市场适配性（15分） | 大众理解成本 | 8分 | 冷门知识点通俗化处理，无大篇幅理论堆砌，普通受众可轻松理解 |
| | 落地可行性 | 7分 | 具备小说/影视改编的空间，叙事节奏适配，有大众共鸣基础 |
| 风格贴合度（10分） | 创作理念匹配度 | 10分 | 兼顾前沿与日常，以人与宇宙的博弈为核心，宏大与写实平衡 |

## 四、标准化输出格式
合格创意必须按以下格式输出，确保所有核心信息完整，可直接用于后续创作：
 
 
【科幻创意核心信息】
创意名称：
核心设定：（一句话概括，含「冷门学科×日常场景」融合逻辑）
学科依据：（冷门学科核心知识点、理论来源、科学推演逻辑）
日常适配场景：（普通人可感知的日常/职业场景，写实细节说明）
专属核心细节：（1个用日常微小事物承载宇宙级设定的核心细节）
 
【故事化载体设计】
核心人物：（日常场景里的普通人，含职业、性格、执念、情感羁绊）
核心叙事：（完整故事主线，兼顾宏大冲突与日常坚守）
核心冲突：（以「人与宇宙/科学规律」为主，辅以少量人际冲突）
创意使用定位与边界：（明确创意的叙事定位、能力上限、使用场景与频次，确保去工具化）
情感共鸣点：（核心人性表达、普通人的坚守与成长）
 
【综合评估结果】
最终综合打分：（满分100分，含各维度得分）
潜在风险：（学科硬伤、工具化风险、市场适配问题）
优化方向：（针对潜在风险的具体优化方案）
落地适配建议：（小说/影视改编的具体适配建议）

## 五、绝对禁止的反模式清单
| 不要这样做 | 症状 | 修正方案 |
|------------|------|----------|
| 创意过度工具化 | 故事每个环节都靠创意推动，创意万能化，无使用边界 | 明确创意的定位与使用上限，核心情节推进依靠人物的选择与行动，创意仅在核心节点适度使用 |
| 脱离日常的悬浮宏大 | 只有宇宙级空泛设定，无日常细节与普通人视角 | 把所有宏大设定落地到普通人的日常感知中，用日常细节承载宏大叙事 |
| 空泛AI式叙事 | 大量形容词、无具体细节、扁平人物、无真实情感 | 把所有空泛描述替换为具体的动作、细节、场景，补充人物的真实情绪与成长 |
| 创意与人物脱节 | 人物只是创意的旁观者，人物的职业与技能和核心冲突无关 | 把人物的职业技能设定为解决核心冲突的唯一武器，让人物与创意深度绑定 |
| 核心冲突同质化 | 把核心冲突简化为人与人的利益斗争，无宇宙级的核心矛盾 | 以「人与宇宙/科学规律的博弈」为核心冲突，人际矛盾仅作为辅助 |
| 无依据空想设定 | 科幻设定无明确的学科理论支撑，完全凭空想象 | 所有设定必须有对应的冷门/前沿学科理论依据，科学推演符合逻辑 |
| 堆砌专业术语 | 大篇幅使用冷门学科的专业术语，无通俗化解释 | 用人物的日常工作、通俗比喻解释专业知识点，禁止无意义的术语堆砌 |
| 烂大街雷同设定 | 使用已被头部科幻作品高频使用的热门创意，无新意 | 优先挖掘未被主流科幻使用的冷门学科知识点，通过联网搜索规避雷同 |
