    How to use

    1. Click Add item to insert slices into the menu.
    2. Select a slice in the list to edit its details and colour.
    3. Use Move up / Move down to reorder slices around the circle.
    4. Export your preset as JSON and drop it into Plasticity's radial menu configuration.

    This project bundles every dependency locally so it can run entirely offline. Save this folder and open index.html in any modern browser.
